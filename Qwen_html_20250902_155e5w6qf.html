How to Share with Sam

Important: This app currently doesn't support real-time collaboration. To share data with Sam:

  1. Copy this code and save it as an HTML file on your computer
  2. Host it on a shared platform like GitHub Pages, Netlify, or Vercel
  3. Or use a collaborative tool like Google Drive to share the file

For true real-time collaboration, you would need a backend server with a database, which is beyond the scope of this implementation.
